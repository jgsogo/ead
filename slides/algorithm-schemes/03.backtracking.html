Búsqueda con retroceso

(Backtracking)

Una búsqueda con retroceso sirve para encontrar la solución óptima (o todas las soluciones posibles) de un problema.

Esquema backtracking

  1. Se comienza con una solución vacía
  2. En cada paso se selecciona uno de los candidatos
  3. Se comprueba si la solución es completable (si añadiendo más se podría llegar a la solución):
    • No: se rechaza el candidato elegido
    • Sí: se añade a esta solución
  4. Si se ha llegado a una solución, se almacena la solución, se elimina el último candidato y se vuelve al paso 2

Búsqueda con retroceso -vs- voraz

Para aplicar búsqueda voraz es necesario poder conocer en cada paso cuál es el mejor elemento.

La búsqueda con retroceso calcula todas las soluciones posibles. Después habrá que elegir la óptima o la mejor.

Ejemplos

  • Problema de las ocho reinas
  • Problema del caballo
  • Atravesar un laberinto
  • Árbol: cálculo de la altura de un árbol
